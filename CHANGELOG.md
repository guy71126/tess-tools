# Changelog

All notable changes to TESS Tools will be documented in this file.

The project follows semantic versioning after the initial alpha release.

## [Unreleased]

### Added

- Cross-platform continuous integration and release automation.
- Public contribution, citation, security, and release documentation.

## [0.1.0] - Unreleased

### Added

- `tess-where` target resolution and TESS sector/product inventory.
- SPOC, QLP, TESS-SPOC, and TESSCut availability detection.
- Science-mode recommendations and machine-readable fetch plans.
- `tess-fetch` downloads, resumable transfers, manifests, and normalized CSV output.
- Provider-specific SPOC, QLP, TESS-SPOC, and TESSCut quality policies.
- TESSCut aperture extraction and diagnostic reports.
- Optional Gaia crowding, TOI, and local TESS eclipsing-binary crossmatches.
- Offline test suite and bounded live validation harness.
